# Cash Me If You Can v2
A narrative-driven financial simulation game where players manage daily expenses while working toward a meaningful goal. [This is a Capstone Project]
Valentines Version

## Overview
Players take on the role of a college student managing a daily allowance over 15 days. Through decision-making and budgeting strategies, players learn how everyday choices impact long-term financial outcomes.

## Features
- Budgeting system (70/30, 50/30/20)
- Daily decision-making loop
- Narrative-driven progression
- Multiple outcomes based on choices
